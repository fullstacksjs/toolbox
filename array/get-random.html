getRandom

Returns a random item from an array

Import

import { getRandom } from '@fullstacksjs/toolbox';

Signature

function getRandom<T>(arr: T[]): T {}

Examples

getRandom([])        // undefined
getRandom([1])       // 1
getRandom([1, 2, 3]) // 1 | 2 | 3
