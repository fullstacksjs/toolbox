# NonEmptyArray

```ts
const arr: NonEmptyArray<string> = ['hello', 'world']; // valid
const arr2: NonEmptyArray<string> = []; // type error
```

An array that **guarantees at least one element**. Useful for functions that require at least one value and to avoid runtime checks for empty arrays.

**Runtime helper:**

```ts
const myArr = asNonEmptyArray(1, 2, 3); // [1, 2, 3]
```
