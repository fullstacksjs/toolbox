# inc

Increments the given number by 1, or by a specified step. Useful for counters and mathematical operations.

### Import

```typescript copy
import { inc } from '@fullstacksjs/toolbox';
```

### Signature

```typescript copy
function inc(n: number, step?: number): number {}
```

### Examples

```typescript copy
inc(3)       // 4
inc(3, 2)    // 5
```
